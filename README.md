# AI Developer Kit

A comprehensive toolkit for AI-assisted development with curated prompts, skills, and workflows.

## Structure

- **.github/** - GitHub-specific configurations and prompts
- **docs/** - Documentation and architecture guides
- **prompt-library/** - Curated prompt templates for various domains
- **templates/** - Project and code templates
- **resources/** - Reference materials and utilities
- **reports/** - Documentation and analysis reports
- **presentations/** - Slide decks and presentations
- **diagrams/** - Architecture and flowchart diagrams

## Quick Start

1. Review the [Architecture](docs/Architecture.md) guide
2. Check the [Roadmap](docs/Roadmap.md) for current initiatives
3. Explore the [Prompt Library](prompt-library/) for domain-specific prompts
4. Follow the [Workflow](docs/Workflow.md) guide
